definition list

(dl , dt , dd)

dl definition list , dt definition title , dd definition description.

Used mostly for Table of Contents or Dictionary Items.

tag be like :

<dl>
          <dt> This is Definition Title </dt>
          <dd> This is Definition Description </dd>
</dl>

ordered list

(ol , li )

ol ordered list , li list Items.

Used for Numbering Items like : Shopping List.

tag be like :

<p> food </p>

<ol>
          <li> Milk </li>
          <li> Eggs </li>
          <li> Bread </li>
</ol>

It will appear like this :

food

  1. Milk
  2. Bread
  3. Eggs

unordered list

(ul , li )

ul unordered list , li list Items.

Used for Bullet points.

tag be like :

<p> Why HTML used for? </p>

<ul>
          <li> Create own Web Pages </li>
          <li> Design Websites </li>
          <li> Show off the skills </li>
</ul>

It will appear like this :

Why HTML used for?

  • Create own Web Pages
  • Design Websites
  • Show off the skills
